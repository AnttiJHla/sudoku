Instructions:

Matrix can be solved automatically step by step by clicking 'Solve' button.
Matrix can be initialized back to original values by clicking 'Initialize matrix'
Original cell values are shown with green color
A new matrix can be selected by clicking one of the buttons under 'Select matrix'
- First round of 'solve' functionality is applied to matrix when it has been selected
- Selecting empty matrix allows you to build your own matrix, it cannot be saved.
Alternatively you can try to solve it manually by clicking the numbers on matrix cells
- Click a value to set value for cell. There is no undo functionality for selecting a value.
- Shift + Click a value to remove a possible value.
